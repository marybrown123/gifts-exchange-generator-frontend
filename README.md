* PostgreSQL
### Frontend
* React.js
* Typescript
* CSS

## Setup

### Backend

* Install all neccessary dependencies:<br/>

        npm install

* Run the server locally:<br/>
   
        npm run dev

* Build for production:<br/>
 
        npm run build
        
### Frontend

* Install all neccessary dependencies:<br/>

        npm install
        
* Run the client locally:<br/>

	    npm run start

* Build for production:<br/>

        npm run build
